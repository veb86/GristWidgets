# Test123 Dashboard Selector Widget

Этот виджет представляет собой селектор дашбордов, который позволяет выбирать и отображать различные внешние дашборды через iframe с настраиваемыми размерами.

## Структура

- **Выпадающий список**: Для выбора дашборда из таблицы данных
- **iFrame область**: Для отображения выбранного дашборда с заданными размерами

## Особенности

- Выбор дашбордов из таблицы данных Grist
- Настраиваемые размеры отображения для каждого дашборда
- Автоматическое добавление параметра `embed=true` к URL
- Адаптивный дизайн: корректное отображение на разных размерах экрана
- Использует Bootstrap CSS для стилизации
- Поддерживает ручное изменение размера
- Взаимодействие с Grist API через `grist-plugin-api.js`

## Требования к таблице данных

Для корректной работы виджета необходимо создать таблицу с именем "dashboards" и следующими колонками:

- `Dashboard_Name`: Название дашборда (строковый тип)
- `Dashboard_Link`: Ссылка на страницу дашборда (URL)
- `Width`: Ширина окна дашборда в пикселях (числовой тип)
- `Height`: Высота окна дашборда в пикселях (числовой тип)

## Технологии

- HTML/CSS/JavaScript
- Bootstrap 5.2.3
- Tabulator Tables
- jQuery
- Grist Plugin API

## Использование

Виджет можно использовать как пользовательский виджет в Grist для отображения внешних дашбордов, панелей управления или других веб-ресурсов с возможностью выбора и настройки размеров отображения.
