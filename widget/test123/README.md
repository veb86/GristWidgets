# Test123 Widget Organizer

Этот виджет представляет собой органайзер с двухколоночным макетом, который позволяет одновременно работать с двумя табличными виджетами: `table1` и `table2`.

## Структура

- **Левая колонка (50%)**: Встраивает виджет `table1` через iframe
- **Правая колонка (50%)**: Встраивает виджет `table2` через iframe

## Особенности

- Адаптивный дизайн: на мобильных устройствах колонки располагаются вертикально
- Использует Bootstrap CSS для стилизации
- Поддерживает ручное изменение размера
- Взаимодействие с Grist API через глобальный объект `AppHost`

## Технологии

- HTML/CSS/JavaScript
- Bootstrap 5.2.3
- Tabulator Tables
- jQuery
- Grist Plugin API

## Использование

Виджет можно использовать как пользовательский виджет в Grist для одновременного просмотра и редактирования данных в двух разных таблицах.
